# NLP Learning Repository

This repository contains a set of hands-on Natural Language Processing (NLP) tutorials and examples, organized as separate modules for key preprocessing and analysis steps.

## Overview

Natural Language Processing enables computers to understand and interpret human language. This repository demonstrates the main stages of an NLP pipeline, including sentence segmentation, tokenization, POS tagging, stemming, lemmatization, stop-word removal, dependency parsing, and named entity recognition.

## Repository Structure

- `intro.txt` - A summary of NLP and the main pipeline steps.
- `pipeline_NLP.png` - A visual diagram of the NLP pipeline.
- `1.sentence_Segmentation/` - Notebook and material for splitting text into sentences.
- `2. Tokenization/` - Notebook and material covering word, character, sub-word, sentence, and n-gram tokenization.
- `3.POS_tagging/` - Notebook and material for part-of-speech tagging.
- `4.1.stemming/` - Introductory material and notebooks for Porter and Snowball stemming algorithms.
- `4.Lemmatization/` - Notebook and material for lemmatization and its advantages.
- `5.Stop words removal/` - Notebook and material for removing common stop words.
- `6.Dependency Parsing/` - Notebook and material for understanding syntactic relationships.
- `7.NER (Named Entity recognition)/` - Notebook and material for identifying names, places, dates, and other entities.

## What You Will Learn

- Why NLP is important and how the NLP pipeline works.
- How to split text into sentences and tokens.
- How to assign part-of-speech tags to words.
- The difference between stemming and lemmatization.
- How stop-word removal improves text processing.
- How dependency parsing reveals grammatical structure.
- How named entity recognition extracts real-world entities from text.

## How to Use

1. Open the repository in Jupyter Notebook or JupyterLab.
2. Explore each folder in order to follow a typical NLP preprocessing workflow.
3. Run the notebooks to see examples and code for each topic.
4. Refer to the `intro.txt` files in each folder for explanations and notes.

## Notes

- This repository is intended as a learning resource and practical guide.
- The notebooks include examples of NLP concepts and common preprocessing techniques.
- Some modules may use Python NLP libraries such as NLTK or spaCy (check the notebooks for exact dependencies).
